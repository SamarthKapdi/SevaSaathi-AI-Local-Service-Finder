# Design Document – SevaSaathi

## Overview
SevaSaathi is an AI-powered local service finder that connects users with nearby service providers like plumbers, electricians, and carpenters. Users can search, book, and rate workers easily.

## Architecture
Client → Frontend → Backend → Database → AI Services

## Modules
- User registration and login
- Worker profile and skill listing
- Service search by location
- Booking and scheduling
- Ratings and reviews
- AI recommendations
- Chatbot support

## Database
Tables: users, providers, services, bookings, reviews

## Tech Stack
Frontend: React  
Backend: Spring Boot  
Database: MySQL  
AI: LLM API  
Hosting: Cloud

## Deployment
Docker containers deployed on cloud servers

## Future Scope
Mobile app, voice support, and digital payments
